## Counter DApp
This app can counter value from initial value 0, for increment by 1 or decrement by 1.

Here are steps for using the app:

1. First thing for using the app is by linking into your coccordium chrome web wallet by click Connect to browser wallet
2. after you are connected, you can start initialize the contract by click Init contract button. then looking for index that you intilize contract.
3. type your index (e.g 3219 for in GIF example) in Contract Index Box, then clik Counter Value button for looking for initial value your counter, should be 0 because by default it was set to 0.
4. you can start click Increment/Decrement button for Uodate the contract by sending transaction to the chain, you counter value will be update +1 for Increment or -1 for Decrement.
5. then you can Invoke counter value in the chain by clik Counter Value again for looking updated counter value.
